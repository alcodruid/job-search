# Идеи статей для Habr.com и vc.ru

Контент-маркетинг как пассивный поиск работы: статьи приносят органический трафик, показывают экспертизу, привлекают работодателей.

---

## Habr.com — технические статьи (AI/DevOps аудитория)

### 1. «Развёртываю qwen2.5:32b дома: реальный опыт private LLM для 6 продуктов»
**Формат:** How-to + опыт  
**Тезисы:** зачем не облако, выбор железа (Tesla P40), Ollama конфигурация, GPU memory для нескольких моделей, реальная нагрузка, чего не хватает в Ollama, что исправил кастомной обёрткой  
**Ожидаемый отклик:** CTO/Head of Engineering у которых стоит задача private AI

### 2. «RAG который работает в продакшне: bge-m3 vs nomic-embed-text на реальных данных»
**Формат:** Сравнительный технический обзор  
**Тезисы:** как выбирал embedding модель для TelecomExpert vs Replyo, разница в качестве поиска, latency, размер индекса, практические советы  
**Ожидаемый отклик:** ML-инженеры, которые строят RAG

### 3. «n8n как оркестратор AI-пайплайнов: что умеет, что не умеет»
**Формат:** Практический обзор  
**Тезисы:** какие AI-задачи делаю в n8n vs в Python, интеграция с Ollama через HTTP, Telegram-нода, работа с JSON из LLM-ответов, ограничения n8n для complex logic  
**Ожидаемый отклик:** DevOps/automation engineers, n8n community

### 4. «19 лет запускал ISP — теперь строю AI: что общего»
**Формат:** Crossover narrative (не строго технический)  
**Тезисы:** операционное мышление в AI, SLA и надёжность инфраструктуры, «никогда не падать» vs «иногда таймаут», управление командой из одного человека (себя), что телеком-CEO не понимал в AI — и что понял  
**Ожидаемый отклик:** широкая IT-аудитория, HR-специалисты, возможные партнёры

## vc.ru — бизнес/предпринимательская аудитория

### 5. «Сколько стоит поднять собственный AI за 2026 год: честные цифры»
**Формат:** Breakdown costs + опыт  
**Тезисы:** железо (б/у Tesla P40 ~30k), электричество, время на настройку, сравнение с OpenAI API по нагрузке, когда окупается  
**Ожидаемый отклик:** предприниматели, CTO малого/среднего бизнеса, потенциальные клиенты

### 6. «Автоматизировал 80% рутины телеком-компании через n8n + LLM: что работает»
**Формат:** Кейс  
**Тезисы:** какие процессы автоматизировал в ТЕЛСИ, что дал LLM (классификация обращений, генерация ответов), ROI, что не поддалось автоматизации  
**Ожидаемый отклик:** операционные директора, малый телеком, любые сервисные компании
